# 介绍
1、语音控制**红外**和**蓝牙**产品
 -  **红外学习功能**，可学习任何遥控器按键
2、4个空调按键，2个窗帘按键
3、4个继电器**开关按键**，或切换为**场景按键**，可以自定义按键绑定
4、蓝牙灯具控制
4、本地场景任务
5、**无需wifi上网**，无需APP，插电即可使用。
## 出厂默认或15秒复位后按键固定为：
1、空调开关：按一下开空调，再按一下关空调
2、空调风量：按一下增加1档风量：自动、低速风、中速风、高速风循环
3、温度+：按一下温度加1度，不循环
4、温度- ：按一下温度减1度，不循环
5、按一下开窗帘，再按一下关窗帘，长按窗帘停止

# 语音控制
1、空调：匹配空调，空调品牌，搜索匹配
2、电视：匹配电视
3、机顶盒：匹配机顶盒
4、风扇：匹配风扇
5、开关：我要匹配->重置开关
6、场景：按键功能切换->场景设置   |||| 删除场景
7、上电触发：场景设置->上电触发->播报欢迎词->场景指令

# 场景联动

语音说“场景设置”→播报“请说场景指令”→说“我回家了” → 播报“请说语音指令” → 说“打开客厅灯”、“打开卧室灯”、“打开电视”、“打开空调”、“打开窗帘”，当完成所有任务设置后，语音说“退出设置”结束场景设置，或者等15秒退出语音唤醒结束场景设置。
